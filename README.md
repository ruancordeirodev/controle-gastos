# Controle de Gastos

Aplicação web para registro e análise de finanças pessoais.

## Funcionalidades
- Cadastro de entradas  e despesas
- Visualização de saldo
- Histórico de movimentações
- Estrutura separada frontend/backend

## Estrutura do projeto

/frontend → interface do usuário (HTML, CSS, JS)
/backend → API e persistência de dados

## Tecnologias
- HTML5, CSS3, JavaScript
- Node.js

## Objetivo
Projeto focado em prática de arquitetura web simples com separação de camadas.
